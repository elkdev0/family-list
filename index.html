📱 Family Sharing Guide

This app saves data on YOUR device only (localStorage). Each family member who opens the file will have their own separate copy. Here are your options:

Option 1 — Share the file (Easy, no sync)
Upload this HTML file to Google Drive. Share the link with family. Each person opens it in their own browser. Everyone manages their own copy. Best for personal tracking only.
Option 2 — Google Sheets (Best for beginners 👍)
Create a Google Sheet with the same columns. Go to Share → add family email → set permission to Editor (so they can update it). Everyone edits the same sheet in real time from any phone. Free, no coding needed.

Permissions: Viewer = can only look. Commenter = can add notes. Editor = can change anything. Give Editor only to trusted family members.
Option 3 — Firebase or Supabase (Best real sync, upgrade later)
These are free cloud databases. With a developer's help (or future Claude session), you can connect this app so all changes sync instantly across every phone. This is the best long-term solution for a family tracker.

💡 Recommendation: Start with Google Sheets for family sharing. Use this HTML file for your own personal tracking. Upgrade to Firebase later when needed.
